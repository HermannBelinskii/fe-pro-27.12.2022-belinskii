• git status - показывает состояние документа
  • git push загружает изменения локально
  • git pull обновляет папку с файлами до последних изменений
  • git branch показывает на какой ветке гита находишься
  • git checkout переключение между ветками
  • git commit сохраняет изменения
  • git add добавление документов
  • git clone скачивание репозитория
